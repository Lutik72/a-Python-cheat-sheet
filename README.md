# ☕ Сайт кофейни

Проект представляет собой многостраничный сайт для кофейни, разработанный на Django.

## ✨ Уже реализовано

### Главная страница
Страница-визитка, знакомящая посетителей с атмосферой и преимуществами заведения.

### Меню
Динамическая страница с карточками позиций меню. Все позиции создаются и редактируются через **админ-панель Django**, что позволяет легко обновлять ассортимент без изменения кода.

### Коворкинг-зона
Отдельная страница с подробным описанием пространства для работы. 
- **Тарифы** полностью управляются через админку — можно добавлять, изменять цены и описания.
- Информация о преимуществах зоны (Wi-Fi, розетки, удобные кресла).

### Форма бронирования
Форма для бронирования мест в коворкинг-зоне.

### Регистрация и вход
Создание аккаунта для доступа к функциям сайта.

### Восстановление пароля
Полноценная система восстановления пароля через email:
- Форма для ввода email зарегистрированного пользователя
- Автоматическая отправка письма со ссылкой для восстановления
- Форма для создания нового пароля
- Страница подтверждения успешного изменения пароля
- Письма отправляются в папку sent_emails (должна быть в корне проекта, добавлена в .gitignore)

### Личный кабинет
Просмотр информации о пользователе.

### 🧪 Тестирование

В проекте написаны тесты для проверки доступности страниц, редиректов и используемых шаблонов.

#### Как запустить тесты?
> ❗ Перед запуском тестов установите зависимости, а также примените миграции

```bash
# запуск всех тестов
python manage.py test

# запуск для конкретного приложения
python manage.py test apps.users

# запуск тестов только представлений
python manage.py test apps.users.tests.test_views

# запуск тестов только ссылок
python manage.py test apps.users.tests.test_urls
```

## Как запустить
1. Клонируйте репозиторий или скачайте его файлы
```bash
git clone https://github.com/Lutik72/Django_Project.git
```
2. Создайте виртуальное окружение и активируйте его
```bash
python -m venv venv # создать

venv\Scripts\activate # активировать
```
3. Установите зависимости
```bash
pip install -r requirements.txt
```
4. Примените миграции
```bash
python manage.py migrate
```
5. Создайте суперпользователя для входа в админку
```bash
python manage.py createsuperuser
```
6. Запустите сервер
```bash
python manage.py runserver
```

### Если возникли ошибки
```bash
# Проверить миграции
python manage.py makemigrations
python manage.py migrate

# Собрать статические файлы
python manage.py collectstatic
```
