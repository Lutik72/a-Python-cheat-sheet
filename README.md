# ☕ Сайт кофейни

Проект представляет собой многостраничный сайт для кофейни, разработанный на Django.

## ✨ Уже реализовано

### Главная страница
Страница-визитка, знакомящая посетителей с атмосферой и преимуществами заведения.

### Меню
Динамическая страница с карточками позиций меню. Все позиции создаются и редактируются через **админ-панель Django**, что позволяет легко обновлять ассортимент без изменения кода.

### Коворкинг-зона
Отдельная страница с подробным описанием пространства для работы. 
- **Тарифы** полностью управляются через админку — можно добавлять, изменять цены и описания.
- Информация о преимуществах зоны (Wi-Fi, розетки, удобные кресла).

## 🚧 В разработке

### Форма бронирования
Интерактивная форма для бронирования мест в коворкинг-зоне. В планах — интеграция с бэкендом для обработки заявок.

## 🛠 Технологии

- **Бэкенд:** Python, Django (админ-панель, шаблоны, работа с данными)
- **Фронтенд:** HTML5, CSS3 (разметка страницы и стили)
- **Иконки:** Font Awesome 6
