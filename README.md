# Знакомство с Python. Переменные и типы данных
Клиент и сервер взаимодействуют в интернете. Бэкенд-разработчик пишет программы для серверов. Бэкенд можно разрабатывать на различных языках программирования, в том числе на Python.

✅ Любая строчка, начинающаяся с символа ```#``` — это комментарий, примечание для разработчика.

## Первая программа
```python
print('Привет, Мир!')  # Традиционная первая программа
```
```
Привет, Мир!
```

## Переменная

✅ **Переменная** – место в памяти, куда сохраняются данные. Любая переменная имеет имя и значение.

✅ Значение переменной — это сохранённая в ней информация: текст, число и т.д.

✅ Именование переменных:
* В имени переменной рекомендуется использовать только латинские буквы a-z, A-Z, цифры и символ нижнего подчеркивания (_).
* Имя переменной не может начинаться с цифры.
* Имя переменной не может содержать пробелы.
* Имя переменной по возможности должно отражать ее назначение.
* Для именования переменных принято использовать стиль `lower_case_with_underscores` (слова из маленьких букв с подчеркиваниями).

✅ **В качестве названия переменных запрещено использовать ключевые слова.**
К ключевым словам в языке Python относятся: `False`, `True`, `None`, `and`, `with`, `as`, `assert`, `break`, `class`, `continue`, `def`, `del`, `elif`, `else`, `except`, `finally`, `try`, `for`, `from`, `global`, `if`, `import`, `in`, `is`, `lambda`, `nonlocal`, `not`, `or`, `pass`, `raise`, `return`, `while`, `yield`.

## Типы данных

### Целые числа
✅ Тип данных `int` используется для представления **целых чисел** 

#### ✅ Создание переменных типа `int`

#### Прямое присвоение целочисленных литералов
```python
# Положительные числа
num1 = 100
# Отрицательные числа
num2 = -500
# Ноль
num3 = 0

# С разделителями для читаемости (используется подчеркивание)
population = 7_800_000_000  
salary = 150_000            
```

#### Создание через арифметические операции
```python
# Результат операции - целое число
result = 10 + 5          # 15 (int)
product = 6 * 7          # 42 (int)
difference = 100 - 50    # 50 (int)
```

#### ✅ Преобразование к типу `int`

#### Из строки
```python
# Преобразование строки, содержащей целое число
num1 = int('12345')      # 12345
num2 = int('-500')       # -500
num3 = int('0')          # 0

# Из пользовательского ввода (input() всегда возвращает строку)
user_input = input("Введите число: ")
number = int(user_input)  # преобразует строку в целое число
```

#### Из числа с плавающей точкой (`float`)
```python
# Отбрасывание дробной части (не округление!)
num1 = int(3.14)     # 3
num2 = int(2.99)     # 2 (не 3!)
num3 = int(-5.9)     # -5 (не -6!)

# Преобразование после математических операций
result = int(10.5 + 2.3)  # 12
```

### Дробные числа (числа с плавающей точкой.)
✅ Тип данных `float` используется для представления **дробных (вещественных) чисел с плавающей точкой**. Это числа, содержащие десятичную дробную часть.

#### ✅ Создание переменных типа `float`
```python
# Прямое присвоение дробных чисел
pi = 3.14159
temperature = -5.5
price = 99.99
zero_point = 0.0
```

#### ✅ Преобразование к типу `float`
```python
# Из строки
num1 = float('3.14')     # 3.14
num2 = float('-2.5')     # -2.5
num3 = float('0.0')      # 0.0

# Из целого числа
num4 = float(100)        # 100.0 (добавляется десятичная часть .0)
num5 = float(-50)        # -50.0
```

## Строки
✅ Строки задаются  одинарными `' '` или двойными `" "` кавычками
```python
word = 'Привет!'
print(word) # Привет!
```
Существует тип строк - `F-строки`

✅  **F-строки** — это специальный тип строк в Python, который позволяет **встраивать выражения и переменные прямо внутрь строки**. 

F-строки начинаются с буквы `f` или `F` перед кавычками:

```python
# Базовая структура
f"Текст {выражение} текст"

# Пример
name = "Анна"
age = 25

message = f"Меня зовут {name} и мне {age} лет"
# Результат: "Меня зовут Анна и мне 25 лет"
```

## Методы для работы со строками

### Метод `split()` — разделение строки на список

```python
text = "один-два-три"
parts = text.split('-')  # ['один', 'два', 'три']
```

### Примеры использования
```python
# Разделение по точке с пробелом
poem = 'Ночь. Улица. Фонарь. Аптека'
words = poem.split('. ')  # Указываем разделитель
print(words)  # ['Ночь', 'Улица', 'Фонарь', 'Аптека']

# Разделение по пробелам (значение по умолчанию)
poem = 'Ночь. Улица. Фонарь. Аптека'
words = poem.split()  # Скобки пустые
print(words)  # ['Ночь.', 'Улица.', 'Фонарь.', 'Аптека']
```

### Метод `join()` — объединение списка в строку

```python
parts = ['один', 'два', 'три']
text = '-'.join(parts)  # 'один-два-три'
```

### Примеры использования
```python
# Создаем список слов
words_list = ['раз', 'два', 'три', 'четыре']

# Объединение с дефисом
result = '-'.join(words_list)
print(result)  # 'раз-два-три-четыре'

# Объединение с пробелом
result = ' '.join(words_list)
print(result)  # 'раз два три четыре'

# Объединение без разделителя
result = ''.join(words_list)
print(result)  # 'раздватричетыре'
```

## Списки
Создание списка (аналог массивов):
```python
fruits = ['яблоко', 'банан', 'апельсин', 'киви', 'манго']
print(fruits)  # ['яблоко', 'банан', 'апельсин', 'киви', 'манго']
```

Количество элементов списка:
```python
count = len(fruits)  # Функция len() подсчитывает элементы
print(count)  # 5
```

## **Основные операции со списками**

### **Добавление элементов**

### `append()` — добавить в конец
```python
fruits = ['яблоко', 'банан']
fruits.append('апельсин')
print(fruits)  # ['яблоко', 'банан', 'апельсин']
```

### `insert()` — вставить на позицию
```python
fruits = ['яблоко', 'банан', 'апельсин']
fruits.insert(1, 'груша')  # Вставляем на второе место (индекс 1)
print(fruits)  # ['яблоко', 'груша', 'банан', 'апельсин']
# Индексы:       0         1        2        3
```

### **Удаление элементов**

### `remove()` — удалить по значению
```python
fruits = ['яблоко', 'банан', 'апельсин', 'банан']
fruits.remove('банан')  # Удаляет только первый 'банан'
print(fruits)  # ['яблоко', 'апельсин', 'банан']
```

## Словари

Словарь (`dict`) оформляется фигурными скобками `{}`. Его заполняют пары "ключ: значение", разделенные двоеточием.

### Создание словаря
```python
# Словарь с описанием питомцев
pets = {
    'кошка': 'Мурка',
    'собака': 'Барсик',
    'хомяк': 'Пушистик'
}

# Словарь с ценами на продукты
prices = {
    'молоко': 80,
    'хлеб': 50,
    'яблоки': 120
}
```

### Доступ к элементам
```python
# Доступ по ключу
print(pets['кошка'])  # Мурка

# Изменение значения
pets['кошка'] = 'Васька'
print(pets['кошка'])  # Васька

# Добавление новой пары
pets['попугай'] = 'Кеша'
```

### Перебор элементов словаря
```python
# По ключам
for key in pets:
    print(f"Вид: {key}, Имя: {pets[key]}")

# По значениям
for name in pets.values():
    print(f"Питомец: {name}")

# По ключам и значениям вместе
for animal_type, name in pets.items():
    print(f"{animal_type} по имени {name}")
```

## Множества

Множество (`set`) похоже на список, но с двумя важными отличиями:
1. Элементы во множестве не повторяются
2. Порядок элементов не гарантируется

### Создание множества
```python
# Множество любимых фруктов
favorite_fruits = {'яблоко', 'банан', 'апельсин', 'банан'}
print(favorite_fruits)  # {'яблоко', 'банан', 'апельсин'} - банан только один раз

# Множество из списка (удаляет дубликаты)
visited_cities = ['Москва', 'Париж', 'Лондон', 'Москва', 'Париж']
unique_cities = set(visited_cities)
print(unique_cities)  # {'Москва', 'Париж', 'Лондон'}
```

### Основные операции
```python
# Проверка наличия элемента
if 'яблоко' in favorite_fruits:
    print("Я люблю яблоки!")

if 'груша' not in favorite_fruits:
    print("А грушу я не люблю")
```

### Операции с множествами
```python
# Два множества фруктов
fruits1 = {'яблоко', 'банан', 'апельсин'}
fruits2 = {'апельсин', 'груша', 'киви'}

# Объединение (все уникальные фрукты из обоих наборов)
all_fruits = fruits1.union(fruits2)
print(all_fruits)  # {'яблоко', 'банан', 'апельсин', 'груша', 'киви'}

# Пересечение (фрукты, которые есть в обоих наборах)
common_fruits = fruits1.intersection(fruits2)
print(common_fruits)  # {'апельсин'}

# Разность (фрукты, которые есть в первом, но нет во втором)
only_in_first = fruits1.difference(fruits2)
print(only_in_first)  # {'яблоко', 'банан'}
```

# Ввод-вывод данных

#### ✅ Для вывода данных на экран используется команда `print()`.
В круглых скобках указываются данные, которые нужно вывести (аргументы). Если это текст, его нужно заключить в одинарные или двойные кавычки. Команда `print()` позволяет указывать несколько аргументов, в таком случае их надо отделять запятыми.

**Аргументы** – это конкретные значения, которые передаются функции при ее вызове.

✅ **Необязательные параметры команды `print()`:**

- `sep` (separator – разделитель) позволяет установить строку, с помощью которой будут разделены аргументы (если их несколько) при печати.
- `end` определяет строку, которая будет добавлена после вывода всех аргументов.

Параметры `sep` и `end` можно устанавливать одновременно.

Символ `\n` называется управляющей последовательностью и задаёт перевод строки.

#### ✅ Для считывания данных в языке Python используется команда `input()`.
Когда программа доходит до места, где есть `input()`, она ожидает, что пользователь введёт данные с клавиатуры и нажмёт Enter.

# Операции над числами

✅ В языке Python, как и в математике, над числами можно совершать 4 основные операции:

| Оператор | Описание     |
|----------|--------------|
| +        | сложение     |
| -        | вычитание    |
| *        | умножение    |
| /        | деление      |

✅ Порядок выполнения операций: первым делом выполняется умножение или деление, затем сложение и вычитание. Для изменения порядка выполнения операций понадобятся скобки.

✅ Оператор `**` используется для возведения числа в степень. Является правоассоциативным (значение выражения вычисляется справа налево) в соответствии с правилами математики. Таким образом, выражение `x ** y ** z` вычисляется как `x ** (y ** z)`.

✅ Оператор целочисленного деления `//` делит одно число на другое и отбрасывает дробную часть результата. При делении отрицательных чисел необходимо помнить, что результат целочисленного деления не превосходит частное. Другими словами, округление всегда берётся в меньшую сторону.

✅ Оператор деления с остатком `%` возвращает остаток от деления двух целых чисел.

✅ Таблица приоритетов арифметических операторов в Python (чем ниже расположен оператор в таблице, тем ниже его приоритет):

| Оператор      | Описание                                                       |
|---------------|----------------------------------------------------------------|
| ()            | Скобки                                                         |
| **            | Возведение в степень                                           |
| -             | Унарный минус                                                  |
| *, /, //, %   | Умножение, деление, целочисленное деление, остаток от деления  |
| +, -          | Сложение, вычитание                                            |

✅ Последняя цифра числа определяется всегда как остаток от деления числа на 10 (`% 10`).

✅ Алгоритм получения цифр n-значного числа `num`:

* **Вариант 1:**
  * Последняя цифра: `num % 10`
  * Предпоследняя цифра: `(num // 10) % 10`
  * Предпредпоследняя цифра: `(num // 100) % 10`
  * ... и т.д

* **Вариант 2:**
  * Последняя цифра: `num % 10`
  * Предпоследняя цифра: `(num % 100) // 10`
  * Предпредпоследняя цифра: `(num % 1000) // 100`
  * ... и т.д

# Операторы сравнения

✅ Оператор присваивания (`=`) присваивает переменным значения.

✅ Для проверки двух элементов на равенство Python использует удвоенный знак равно (`==`).

✅ В Python существует 6 основных операторов сравнения.

| Выражение   | Описание                         |
|-------------|----------------------------------|
| `x > 7`     | если `x` больше 7                |
| `x < 7`     | если `x` меньше 7                |
| `x >= 7`    | если `x` больше либо равен 7     |
| `x <= 7`    | если `x` меньше либо равен 7     |
| `x == 7`    | если `x` равен 7                 |
| `x != 7`    | если `x` не равен 7              |

# Логические операторы

✅ В Python есть три логических оператора, которые позволяют создавать сложные условия:
* `and` — логическое умножение;
* `or` — логическое сложение;
* `not` — логическое отрицание.

✅ Таблица истинности для оператора `and`:

|   a   |   b   | a and b |
|:-----:|:-----:|:-------:|
| false | false | false   |
| false | true  | false   |
| true  | false | false   |
| true  | true  | true    |

✅ Таблица истинности для оператора `or`:

|   a   |   b   | a or b |
|:-----:|:-----:|:-------:|
| false | false | false   |
| false | true  | true    |
| true  | false | true    |
| true  | true  | true    |

✅ Таблица истинности для оператора `not`:

|   a   | not a |
|:-----:|:-----:|
| true  | false |
| false | true  |

✅ Приоритет выполнения логических операторов:
1. В первую очередь выполняется логическое отрицание `not`;
2. Далее выполняется логическое умножение `and`;
3. Далее выполняется логическое сложение `or`.

**Для явного указания порядка выполнения** условных операторов **используют скобки.**

# Числовые типы данных: `max`, `min`, `abs`

✅ Функции `min()` и `max()` используются для определения минимального или максимального значения соответственно.

**Примеры использования `min()` и `max()`:**

```python
# Для списка чисел
numbers = [5, 2, 8, 1, 9]
print(min(numbers))  # Вывод: 1
print(max(numbers))  # Вывод: 9

# Для отдельных аргументов
print(min(10, 3, 7, 15))  # Вывод: 3
print(max(10, 3, 7, 15))  # Вывод: 15
```

✅ Функция `abs()` используется для определения модуля числа.

✅ **Модуль числа** — это само это число. Модуль (абсолютная величина) показывает величину числа без учёта его знака.

**Примеры использования `abs()`:**

```python
# Положительные числа
print(abs(5))      # Вывод: 5
print(abs(10.5))   # Вывод: 10.5

# Отрицательные числа
print(abs(-3))     # Вывод: 3
print(abs(-7.8))   # Вывод: 7.8

# Ноль
print(abs(0))      # Вывод: 0

# Использование в выражениях
result = abs(-15) + abs(10)
print(result)      # Вывод: 25
```

# Условный оператор if-else

✅ **В Python проверка условия осуществляется при помощи ключевого слова `if`.**

```
if условие:
    блок кода
else:
    блок кода
```

✅ Если требуется проверить несколько условий, в языке Python используется **каскадный условный оператор `elif`.**

✅ Синтаксис каскадного условного оператора имеет следующий вид:

```
if условие1:
    блок кода
elif условие2:
    блок кода
else:
    блок кода
```

✅ При исполнении такого условного оператора сначала проверяется условие1. Если оно является истинным, то исполняется блок кода, который следует сразу после него, вплоть до выражения `elif`. Остальная часть конструкции игнорируется. Однако если условие1 является ложным, то программа перескакивает непосредственно к следующему выражению `elif` и проверяет условие2. Если оно истинное, то исполняется блок кода, который следует сразу после него, вплоть до следующего выражения `elif`. И остальная часть условного оператора тогда игнорируется. Этот процесс продолжается до тех пор, пока не будет найдено условие, которое является истинным, либо пока больше не останется выражений `elif`. Если ни одно условие не является истинным, то исполняется блок кода после выражения `else`.

# Функции

**Функции в Python** — это блоки кода, которые выполняют определенную задачу. Они позволяют структурировать код, избегать повторений и делать программу более читаемой.

## Создание функции

Для создания функции используется ключевое слово `def`:

```python
def say_hello():
    print("Привет!")
```

## Вызов функции

Чтобы использовать функцию, её нужно вызвать по имени:

```python
say_hello()  # Выведет: Привет!
```

## Аргументы функции

Аргументы — это значения, которые передаются функции:

```python
def say_hello(name):
    print(f"Привет, {name}!")

say_hello("Анна")  # Привет, Анна!
```

## Возврат значений из функции

Для возврата значения используется `return`:

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8
```

### Как работает `return`

1. **Завершает выполнение функции:**
```python
def check(num):
    if num > 0:
        return "Положительное"
    return "Не положительное"

print(check(5))  # Положительное
```

2. **Можно вернуть несколько значений:**
```python
def get_info():
    return "Алексей", 30

name, age = get_info()
print(f"Имя: {name}, Возраст: {age}")
```

3. **Без `return` возвращает `None`:**
```python
def say_hello():
    print("Привет!")

result = say_hello()  # Привет!
print(result)         # None
```

# Django
## Что такое Django?
**Django** — высокоуровневый Python-фреймворк с открытым кодом для быстрой и безопасной разработки веб-приложений. "Для перфекционистов с дедлайнами."

### Архитектура MVT
Django использует шаблон **Model-View-Template**:

| Компонент | Описание |
|-----------|----------|
| **Model** | Работа с БД, структура данных |
| **View** | Логика обработки запросов, что показать |
| **Template** | HTML-шаблоны, внешний вид |

> 💡 Она разделяет внутреннюю логику работы сайта, внешний вид страниц для пользователя и реакции веб-сервиса на внешние воздействия.

## Представления (Views)

Файл `views.py` содержит логику обработки запросов.

```python
from django.http import HttpResponse
from django.shortcuts import render

# Простое представление
def simple_view(request):
    return HttpResponse("Привет, мир!")

# Представление с шаблоном
def home_view(request):
    context = {
        'user_name': 'Соня',
        'weather': 'Пасмурно'
    }
    return render(request, 'home.html', context)
```

> 🎯 **render()** — функция-сокращение для отрисовки шаблона с контекстом.

## Маршрутизация (URLs)

from django.contrib import admin

## Модели в Django 
### Что такое модель?

**Модель** — это Python-класс, который описывает структуру таблицы в базе данных. Наследуется от django.db.models.Model.

```python
from django.db import models

class Product(models.Model):
    name = models.CharField(max_length=100)
    price = models.DecimalField(max_digits=10, decimal_places=2)
```

> 📦 Простая аналогия: Класс = таблица, атрибуты = столбцы, экземпляр = строка.

### Миграции
```bash
# После создания/изменения моделей:
python manage.py makemigrations    # создаем миграции
python manage.py migrate           # применяем к БД
python manage.py createsuperuser   # создание админа, для в входа в админку
```
> ⚠️ Важно: Команда **createsuperuser** работает только после применения миграций (migrate), так как создает записи в таблицах аутентификации.



### Файл `admin.py`
```python
from django.contrib import admin
from .models import Product

# Чтобы сделать модель доступной для управления через админку, ее необходимо зарегистрировать.

admin.site.register(Product) #Простейший способ регистрации модели
```
Чтобы сделать админку доступной по определенному URL, нужно настроить маршрутизацию в главном файле urls.py вашего проекта. 
```python
path('admin/', admin.site.urls), # Эта строка включает URL админки
```
### Что получилось?
 В админке Django появится: Раздел Products (Товары)

Теперь через админку можно:
- ➕ Создавать новые товары
- ✏️ Редактировать существующие
- 🔍 Просматривать список
- 🗑️ Удалять товары
